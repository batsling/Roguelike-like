🔴 Browser Cache Test

Instructions:

  1. If you see this page with a RED DASHED BORDER around the box below, CSS is loading correctly
  2. If you don't see red borders, your browser is aggressively caching CSS files
  3. Try these cache clearing methods in order:

Method 1: Hard Refresh

  • Windows/Linux: Ctrl + F5 or Ctrl + Shift + R
  • Mac: Cmd + Shift + R

Method 2: Clear Browser Cache

  • Chrome: Settings → Privacy → Clear browsing data → Cached images and files
  • Firefox: Options → Privacy → Cookies and Site Data → Clear Data
  • Edge: Settings → Privacy → Clear browsing data

Method 3: Disable Cache in DevTools

  • Press F12 to open DevTools
  • Go to Network tab
  • Check "Disable cache"
  • Keep DevTools open while testing

Method 4: Private/Incognito Window

  • Open a new private/incognito window
  • Navigate to the game URL
  • This bypasses all cache
